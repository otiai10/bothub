BotHub API

# Add queue

botによる通知を登録します.

Endpoint

POST http://bothub.otiai10.com/api/queue/add

Parameters

nametype (example)description
user string ("otiai10") 通知をうけるマスターのTwitterアカウント名です. このマスターに紐づいたbotが登録されていないとエラーを返します.
finish int64 (1402820316) 通知をうける時刻のUnixタイム(秒)です. サーバが受け付けた時刻よりも前の時刻の場合、エラーを返します.
text string ("パンツをはいて外に出る時間です") 通知ツイートの内容です. 100文字以上の場合エラーを返します. なお、/@[a-zA-Z0-9_]+/な文字列はマスター以外のユーザへのメンションと判定され、@をのぞいた形でツイートされます.
